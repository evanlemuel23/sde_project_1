Eligibility Criteria

B.Tech (UG) Programs

  • ✓ 12th pass (PCM/PCB) with min 50% marks
  • ✓ Valid JEE Main/AIEEE/CUSAT CAT score
  • ✓ KCET or state entrance exam scores
  • ✓ Merit-based admission available
  • ✓ Age limit: No upper limit
  • ✓ All Indian nationals & NRI citizens eligible

M.Tech (PG) Programs

  • ✓ B.Tech/BE degree with min 60% aggregate
  • ✓ Valid GATE/GPAT score (preferred)
  • ✓ Merit-based admission available
  • ✓ Relevant work experience beneficial
  • ✓ CSE/ECE/ME/CE specializations offered
  • ✓ Research-focused curriculum
